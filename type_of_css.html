CSS

CSS stands for Cascaded Style Sheet and it is used to design webpages ,We can apply CSS in three ways in any HTML markup 
    1.Inline CSS
    2.Internal CSS
    3.External CSS
In Inline CSS , we use style attribute to add design to single HTML element.
If we need to apply same properties for multiple elements, we need to use same properties for all the elements Syntax: "property :value"; Note: If we are applying multiple css properties , al the properties should terminate with semicolon mandatorily but for the last CSS properties, semicolon is not mandatory.

In internal CSS to apply designs we need to target the HTML tags by using selectors and combinators. We use style tags to apply designs in headof html document. Internal CSS
selecting a html tags in a external css is same as internal css but here we use .css extension files to apply css designs. External CSS
