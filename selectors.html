Id Selector

In an ID SELECTOR we will target the html elements having id as the attribute. To represent the id we use # in css file or style tag . Id attribute should be unique for all the html elements so we use this Id selector only for the specified portions of web pages like footers, headers, navigation bars, etc



Universal Selector

In Universal Selector, we can target all the html elements by using * as a selector, this selector willl apply the css properties for all the tags irrespective of tag name.



Attribute Selector

Here we will select the required HTML tags by using tag names with the attribute name and value. Syntax =tagname[attribute="vlaue"]{ properties } We can use this attribute selectors in group



Generic Selector

We will target the html elements while using id's and classes. Syntax : tagname #idname { propeties } tagname.classname{ properties }
